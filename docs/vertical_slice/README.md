# 《雨灯镇：第十三封来信》垂直切片总览

## 项目定位

《雨灯镇：第十三封来信》是一款 5-8 小时体量的中文精致像素 2D RPG。玩家扮演失忆的死信邮差阿澄，在常年下雨的雨灯镇投递十三封本不该存在的信。垂直切片聚焦序章与第一封信《桥边等信的人》，目标时长 35-45 分钟。

## 一句话卖点

在一座永远下雨的小镇里，玩家用“拆封真相”的回合制战斗，把不肯离开的遗憾寄往该去的地方。

## 已锁定方向

- 游戏语言：中文
- 引擎目标：Godot 4
- 规模：5-8 小时完整游戏，先做 35-45 分钟高质量垂直切片
- 美术：精致 3/4 俯视像素 RPG，冷雨暖灯对比
- 叙事：温柔但有后劲，克制、生活化、有余味
- 战斗：无随机遇敌，轻策略回合制，核心为“拆封三层真相”
- 成长：取消传统经验等级，使用“邮差手册”成长
- UI：旧邮册、信封、邮戳、蜡封、雨灯视觉语言

## 垂直切片体验目标

1. 前 5 分钟让玩家记住雨灯镇的雨、灯和旧邮局。
2. 15 分钟内建立阿澄与祈的互动魅力。
3. 第一场 Boss 战让玩家明白“战斗不是击杀，而是读懂一个人”。
4. 结尾出现第十三封信，让玩家愿意继续玩完整版本。

## 垂直切片流程

1. 开场：阿澄撑伞抵达雨灯镇，邮包里有 13 封湿信。
2. 旧邮局：遇见铜雨灯里的祈，获得邮差手册。
3. 第一封信：收件人温衡，地址为雨灯镇旧石桥。
4. 雨灯街探索：询问纸伞铺、面包店、小孩、巡夜人。
5. 石桥初见：温衡接信但拒绝拆开。
6. 面包店调查：发现林茉多年替温衡儿子代写来信。
7. 记忆回溯：看见温衡反复写下未寄出的告别。
8. Boss 战：与“无址回信”对抗，拆开三层封缄。
9. 收束：温衡把信托付给阿澄，第一封信寄出。
10. 钩子：第十三封信露出“澄”字，梁叔在旧钟楼下现身。

## 高好评关键

- 每个系统都必须服务“投递遗憾”这个主题。
- 每场战斗都要有剧情意义，不能变成普通换皮回合制。
- 每 5 分钟给玩家一个新发现：新角色、新疑点、新机制或新情绪。
- 像素资产先做阿澄、祈、无址回信三个标杆，再扩展地图和 NPC。
- 中文台词避免大段设定解释，靠调查物、NPC 反应和地图变化拼真相。
