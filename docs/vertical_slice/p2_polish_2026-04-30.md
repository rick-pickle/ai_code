# P2 精修记录 2026-04-30

## 本轮目标

当前版本已按 90 分线通过，本轮继续做 P2 精修，目标是把垂直切片从“可通过”提升到更像完整、高质量 2D 像素 RPG。范围限定为观感、稳定性和引导一致性，不新增剧情系统，不大规模重构。

## 改动摘要

- 战斗 UI 舞台化：把技能区包成独立“投递指令”指令栏，强化敌方面板、封缄、日志和反馈区域的纸张/封缄层次；保持 960x540 与最大化窗口下关键元素完整可见。
- DialogueLayer 长文本/多选项保护：正文与选项区改为 ScrollContainer；多选项时对话框临时扩高并限制选项滚动区域，避免按钮或文本被推到屏幕外。
- 主线 marker 一致性：在原有 `v 手册` 基础上补充 `v 湿信`、`v 桥灯`、`v 信箱`、`v 信纸`、`v 水洼` 等轻量提示；marker 严格按 flag 阶段显示，完成后消失。
- 窗口完整走查退出清理：`godot_visual_playthrough.gd` 退出前释放运行期 GameRoot/Boot、释放输入状态、停止 AudioManager 并等待若干帧，消除了 ObjectDB leak warning。
- 截图脚本增强：`godot_capture_experience_screens.gd` 新增长文本、多选项、marker 出现/消失截图与布局检查输出。

## 剩余风险

- DialogueLayer 已有滚动保护，但如果未来加入非常长的多段分支对话，仍建议进一步做显式分页和“滚动到底后继续”的输入规则。
- 战斗 UI 已从信息面板向战斗舞台推进，但还没有真正的背景粒子/敌方入场演出；后续若冲更高完成度，可在不影响可见性的前提下补轻量入场和破封特效。
- Marker 当前覆盖主线关键节点，支线或可选调查点仍保持探索式提示。
